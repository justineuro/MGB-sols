This repository (MGB-sols) contains a partial solutions manual to Introduction to the Theory of Statistics (1974) by A.M. Mood, F.A. Graybill, and D.C. Boes.

For the impatient …

A link to a PDF copy of the partial solutions manual or zipped repository:   PDF    MGB-sols-main.zip

To create your own PDF copy

To create your own PDF copy, clone this repository, open the file mgb-main.tex in TeXstudio (or a similar software), then compile it.

Alternatively, clone this repository, then at the linux command line under your local copy of MGB-sols folder (or MGB-sols-main folder if you downloaded a copy of the zipped repository), execute pdflatex mgb-main.tex. That is,

git clone https://github.com/justineuro/MGB-sols
cd MGB-sols
pdflatex mgb-main.tex
